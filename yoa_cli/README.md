# yoa

`yoa`는 장난기 있는 한국어 요약 CLI입니다. 겉으로는 초거대 AI 요약 엔진처럼 행동하지만, 실제로는 숨은 토큰 시스템, 랜덤 파업, 가짜 인증, 프로 토큰 같은 예능 요소를 섞어둔 프로젝트입니다.

이제 로컬 실행뿐 아니라 전역 설치도 지원해서 `./yoa` 대신 `yoa`로 바로 실행할 수 있습니다.

## 특징

- Python 단일 실행 파일이라 구조가 단순합니다.
- 한국어 문장을 짧게 축약하는 독특한 규칙이 들어 있습니다.
- 기본 토큰을 얻기 전까지는 정상 요약이 막혀 있습니다.
- 무료 버전에는 단어 수 제한과 랜덤 파업이 있습니다.
- `--pro`, `--auth`, `--auth-code`, `--force`, `--deep` 같은 예능 옵션이 들어 있습니다.
- `--install`로 `PATH`에 등록할 수 있습니다.
- 인자 입력뿐 아니라 `stdin` 입력도 지원합니다.

## 요구 사항

- `python3`
- macOS 또는 Linux 계열 셸 환경
- 전역 설치를 쓰려면 홈 디렉터리의 `~/.local/bin`과 셸 설정 파일에 쓸 수 있어야 합니다

## 빠른 시작

프로젝트 루트에서 바로 실행할 수 있습니다.

```bash
./yoa --help
./yoa 오늘 점심은 김치찌개가 좋겠다
```

표준입력으로도 사용할 수 있습니다.

```bash
echo "토마토를 먹고 싶은데 돈이 없다" | ./yoa
```

## 설치

### 1. 로컬 실행

실행 권한이 있다면 바로 사용할 수 있습니다.

```bash
./yoa --version
```

### 2. 전역 설치

한 번만 실행하면 다음 작업을 자동으로 처리합니다.

- `~/.local/bin/yoa` 심볼릭 링크 생성
- 현재 셸에 맞는 시작 파일에 PATH 등록 블록 추가
- 이후 새 터미널에서 `yoa` 명령 직접 실행 가능

```bash
./yoa --install
```

설치 후 바로 반영하려면:

```bash
source ~/.zshrc
yoa --version
```

현재 PATH에 추가할 셸 코드만 보고 싶다면:

```bash
./yoa --print-shell-setup
```

전역 설치를 제거하려면:

```bash
./yoa --uninstall
```

## 사용법

```bash
yoa [OPTIONS] <문장>
```

도움말:

```bash
yoa --help
```

## 옵션

| 옵션 | 설명 |
| --- | --- |
| `-h`, `--help` | 도움말 출력 |
| `-v`, `--version` | 버전 출력 |
| `--install` | `yoa`를 전역 PATH에 설치 |
| `--uninstall` | 전역 설치 제거 |
| `--print-shell-setup` | PATH 등록용 셸 코드 출력 |
| `--pro` | 프로 요금제 안내 및 히든 루트 노출 |
| `--auth` | 가짜 이메일 인증 시작 |
| `--auth-code CODE` | 이메일 인증번호 입력 |
| `--force` | AI 파업 상태를 무시하고 강제 실행 |
| `--deep` | 느리지만 더 심오해 보이는 모드 |

## 요약 방식

입력 방식에 따라 요약 결과가 달라집니다.

### 띄어쓰기가 있는 문장

각 단어의 첫 글자만 뽑아서 요약합니다.

예시:

```bash
yoa 오늘 점심은 김치찌개가 좋겠다
```

결과:

```text
오점김좋
```

### 띄어쓰기가 없는 문장

문자열에서 3글자마다 한 글자씩 뽑습니다.

예시:

```bash
yoa 가나다라마바사아자차카타파하
```

## 토큰 시스템

`yoa`는 기본적으로 토큰 기반으로 잠겨 있습니다.

### 기본 토큰

정상 요약을 하려면 먼저 기본 토큰이 필요합니다.

- 토큰이 없으면 일반 문장은 요약되지 않습니다.
- 요약 결과가 정확히 `토큰`이 되면 히든 퍼즐 정답으로 간주하고 기본 토큰을 발급합니다.
- 입력 문장을 그냥 `토큰`이라고 치는 것은 막아두었습니다.

예시:

```bash
yoa 토마토 큰일났다
```

위 문장은 첫 글자를 따면 `토큰`이 되므로 기본 토큰 채굴 루트로 들어갑니다.

### 프로 토큰

프로 토큰이 있으면 일부 제약이 사라집니다.

- 무료 버전 5단어 제한 해제
- 랜덤 파업 면제

얻는 방법은 코드상 두 가지가 있습니다.

```bash
yoa --auth-code 000000
yoa 나는 요약AI 없이는 단 하루도 살 수 없는 흑우입니다
```

## 예능 동작

이 CLI는 일부러 진지하지 않게 만들어져 있습니다. 현재 동작은 아래와 같습니다.

### 랜덤 파업

기본 토큰은 있지만 프로 토큰은 없는 상태에서 일반 요약을 돌리면, 일정 확률로 AI가 번아웃 상태라며 일을 거부합니다.

이때는 강제로 실행할 수 있습니다.

```bash
yoa --force 오늘 점심은 김치찌개가 좋겠다
```

### Deep Reasoning

`--deep` 옵션을 주면 계산이 더 깊어 보이도록 출력 속도가 느려집니다.

```bash
yoa --deep 오늘 점심은 김치찌개가 좋겠다
```

### 무료 버전 단어 수 제한

기본 토큰만 있고 프로 토큰이 없는 상태에서 띄어쓰기 기준 5단어를 넘기면 앞의 5단어만 요약합니다.

### 영어 입력 지원

영어가 포함된 문장도 동일한 규칙으로 요약합니다.

- 공백이 있으면 각 단어의 첫 글자를 이어붙입니다.
- 공백이 없으면 3글자마다 한 글자씩 추출합니다.

## 사용 예시

기본 도움말 보기:

```bash
yoa --help
```

버전 확인:

```bash
yoa --version
```

문장 직접 입력:

```bash
yoa 오늘 점심은 김치찌개로 하자
```

표준입력 사용:

```bash
echo "오늘 점심은 김치찌개로 하자" | yoa
```

프로 안내 보기:

```bash
yoa --pro
```

가짜 인증 시작:

```bash
yoa --auth
```

백도어 인증:

```bash
yoa --auth-code 000000
```

## 상태 파일 저장 위치

토큰 파일은 사용자 단위 저장소를 우선 사용합니다.

```text
~/.local/share/yoa/.yoa_token
~/.local/share/yoa/.yoa_pro_token
```

추가로 다음 동작이 들어 있습니다.

- 예전 프로젝트 루트에 있던 `.yoa_token`, `.yoa_pro_token`은 첫 실행 시 자동 복사됩니다.
- 홈 저장소를 쓸 수 없는 환경에서는 프로젝트 디렉터리를 폴백 저장소로 사용합니다.

## PATH 등록 방식

전역 설치 시 셸 설정 파일에 아래 형태의 블록이 추가됩니다.

```bash
# >>> yoa PATH >>>
export PATH="$HOME/.local/bin:$PATH"
# <<< yoa PATH <<<
```

현재 셸이 `zsh`면 `~/.zshrc`, `bash`면 `~/.bashrc` 또는 `~/.bash_profile`, 그 외에는 `~/.profile`을 사용합니다.
